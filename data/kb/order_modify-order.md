# Modifying an order before it ships

You can edit the shipping address, contact phone and gift message of an order as long as its status is still "Processing". After that, the warehouse picks the items and changes are no longer possible.

To edit, open the order and click "Edit shipping details" or "Edit gift message". Adding or removing items from a placed order is not supported; if you need to change quantities or products, cancel the order and place a new one — the cancelled order's refund is processed in 3 to 5 business days.

For urgent address changes already in shipped status, contact the carrier directly using the tracking link.
